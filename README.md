# MY-CINEMA
My Cinema is an administration dashboard for managing a cinema. This website was created by me as part of the Epitech School project. I helps me to understand about PHP Object-Oriented Programming, MVC Architecture, CRUD Logic, Database Relationships,  PDO as Security Basic, etc.   



## Installation
- PHP 8+
- MySQL
- Local server (MAMP / XAMPP / WAMP)

Place the project in your server root:

htdocs/My_Cinema_Webacademie/

---

## Database
1. Start MySQL
2. Import:

Database/script.sql


Database config file:

Back-End/config/database.php

Default credentials:
- host: 127.0.0.1
- database: my_cinema
- user: root
- password: (empty)

---

## Launch
Start your local server, then open:

http://127.0.0.1:8888/My_Cinema_Webacademie/Front-End/index.html

---

## Notes
- Backend: PHP (MVC, PDO)
- Frontend: HTML, CSS, JavaScript
- Communication via JSON API (`fetch`)
